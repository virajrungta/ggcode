# Green Genius Project

Welcome to the Green Genius project. This repository contains the source code for the Green Genius ecosystem, including mobile, web, and AI components.

## Project Structure

- **[mobile/](file:///Users/virajrungta/Desktop/GreenGenius_Code/greenGenius/mobile)**: React Native (Expo) based mobile application for plant identification and sensor monitoring.

## Getting Started

### Mobile

Navigate to the `mobile/` directory and run:

```bash
npm install
npx expo start
```
